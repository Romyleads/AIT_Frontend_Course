Категория Java JavaScript Сходства Различия
Примитивные
типы		 byte, short, int, long number Обе имеют числовые, логические, символьные типы, null/undefined-значения В Java есть разделение на целые типы; в JS – все числа (целые и дробные) — один тип
float, double В Java есть типы для чисел с плавающей точкой; в JS — number и с 2020 года BigInt
char string (все строки, символа отдельного нет) В Java char — отдельный тип, в JS строки (string) для всего
boolean boolean Тип совпадает; значения: true/false
void undefined Используются как 'пустой' тип результата, но в Java — только для методов, в JS — тип значения
null null, undefined В JavaScript есть и null, и undefined; в Java — только null как значение отсутствия объекта
BigInteger,
BigDecimal		 BigInt Для работы с очень большими числами: в Java BigInteger/BigDecimal, в JS — BigInt
- symbol symbol — только в JS, уникальные идентификаторы
Обёртки
(Wrapper-тип)		 Integer, Double, Float, Boolean, Character, Long, Short, Byte Number, Boolean, String, Symbol, BigInt Обеспечивают объектную оболочку для примитивов В Java больше классов-обёрток по отдельности на каждый тип. В JS — единые классы для number/boolean/string
String (immutable) String (immutable) Строки — неизменяемые объекты В Java — отдельный класс, в JS — встроенный тип, тоже неизменяем
— Object (обёртка для любого примитива) В JS можно обернуть любой примитив в объект вручную В Java такого механизма нет
Коллекции Array, ArrayList, LinkedList, HashSet, TreeSet, HashMap, TreeMap, Queue, Stack и др. Array, Set, Map, WeakSet, WeakMap, Object Есть массивы и коллекции В Java строго типизированные generics-коллекции, в JS — динамическая типизация; коллекции появились позже и чаще всего используются массивы и объекты
Enum (enum MyEnum {A,B,C}) Нет явного enum, только имитация объектом или типа symbol Можно перечислять фиксированный набор значений В Java enum — отдельная категория, в JS — только эмуляция
Объекты Любой объект класса Object, функция как объект, класс как функция Всё можно представить в виде объектов (ООП) В JS прототипное наследование и всё — объекты, в Java пример — классическое ООП
